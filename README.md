# Inventory Management System

A full-stack inventory management application with .NET Core API backend and Angular 18+ frontend.

## Features

- 🔐 User authentication with JWT tokens
- 👥 Role-based access control (Admin/User)
- 📦 Product management (CRUD operations)
- 🏷️ Category management (CRUD operations)
- 📊 Dashboard with real-time statistics
- 📤 Bulk product/category upload
- 🎨 Responsive design with Bootstrap
- 👁️ Password visibility toggle
- 🛡️ Admin panel for bulk operations
- 🚪 Multiple logout options

## Tech Stack

### Backend
- .NET Core 8.0
- Entity Framework Core
- SQL Server / LocalDB
- JWT Authentication
- AutoMapper
- Swagger/OpenAPI

### Frontend
- Angular 18+
- Bootstrap 5
- Font Awesome
- RxJS
- Angular Router

## Prerequisites

- Node.js (v18 or higher)
- .NET Core SDK 8.0
- SQL Server or LocalDB
- Angular CLI 18+

## Installation

### Clone the repository
```bash
git clone https://github.com/aarmah/InventoryManagementSystem.git
